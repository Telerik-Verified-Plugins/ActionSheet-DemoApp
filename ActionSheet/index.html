ActionSheet Plugin Demo

The ActionSheet plugin shows a native list of options the user can choose from. Your code will receive the button index of the option the user has chosen.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS/Android- App package.

ActionSheet

On iOS you'll always see a cancel button. On Android and WP8 you can choose to not show it because the hardware backbutton also hides the ActionSheet.
